管理员： 基本功能包括登录、注册和退出，提供主页导航、轮播图和用户管理功能。支持废品类型及信息管理，公告信息发布与编辑，系统管理与投诉处理。用户信息可查看与修改，并具备权限管理及登录用户状态提示功能。
